# Expense Tracker

A web-based expense tracker built with Dash.

## Setup

1. Clone the repository
    ```bash
    git clone https://github.com/Prot10/money-tracker.git
    ```
2. Install the dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. Run the application:
    ```python
    python run.py
    ```
